# Forest

A high-fidelity, procedural spruce forest simulation. <br>
Rendered using UTF-8 Braille characters in the terminal. It's just cozy and cool yk.
## Example
> [!IMPORTANT]
> This project is currently at it's alpha version as of [v0.0.4](https://github.com/dummy3ye/forest/releases/tag/v0.0.4), for windows you just can install the `.exe` file and give it a go. It recent update it will be available in an alternate `iwr` version with a cli `forest` command.

<img src="assets/forest.gif" alt="Forest" />

## Features
- **Needle Texturing**: Sub-pixel detail for realistic foliage.
- **Somewhat proper Ground Rendering**: Smooth depth effects and stabilized terrain.
- **Seeded Generation**: Deterministic and gnarled tree structures. (keeps every run diffrent and distinctly diffrent)

> The raw c code might comtain bugs and problems feel free to create an issue or create a pull request.
