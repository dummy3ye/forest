# Forest

A high-fidelity, procedural spruce forest simulation rendered using UTF-8 Braille characters in the terminal.

## Features
- **Procedural Needle Texturing**: Sub-pixel detail for realistic foliage.
- **Dynamic Chunk Pooling**: Infinite, memory-efficient world generation.
- **Parallax Ground Rendering**: Smooth depth effects and stabilized terrain.
- **Seeded Generation**: Deterministic and gnarled tree structures.
