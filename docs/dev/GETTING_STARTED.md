## Installation

First clone the github repo using `git`

```bash
git clone https://github.com/dummy3ye/forest.git
cd forest
```

## Project Structure
- `src/`: Core implementation logic (.c files).
- `include/`: Header files (.h files).
- `assets/`: Images and static resources.
- `docs/`: Technical documentation and design notes.
- `dist/`: final binaries and executable files for distribution.
- `build/`: Compiled binaries and object files.

## Building and Running
To build the project, ensure you have `gcc` and `make` installed.

```bash
make
make run
```
